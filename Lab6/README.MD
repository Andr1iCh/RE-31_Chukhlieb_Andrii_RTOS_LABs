Лабораторна робота 6 Широтно-імпульсна модуляція для управління силовими
елементами на основі апаратних таймерів
Обладнання: STM32F411 Discovery, CubeMX, Keil.
Мета: Ознайомитися з принципами роботи широтно-імпульсної модуляції (ШІМ) на
STM32F411, навчитися використовувати апаратні таймери для керування силовими елементами
та реалізувати програму з керуванням шириною імпульсу за допомогою кнопки.

Варіант 3 (23)
